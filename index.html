Activer la communauté (Supabase)

  1. Créez un projet sur supabase.com (offre gratuite).
  2. SQL Editor : collez supabase-camions.sql puis Run.
  3. Project Settings › API : copiez l'URL et la clé anon public.
  4. Renseignez-les en haut du fichier HTML :
const SUPABASE_URL  = "https://xxxx.supabase.co";
const SUPABASE_ANON = "eyJhbGciOiJ...";

Contributions & photos

Lorsque vous ajoutez un camion, les informations saisies (nom, emplacement, jours, horaires, téléphone éventuel) et les photos sont enregistrées via Supabase. Elles sont relues et modérées, puis visibles publiquement. Ne publiez que des contenus appropriés que vous êtes en droit de partager.
